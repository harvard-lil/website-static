Sketch: Local Memory

Where can I find Local Memory?

http://www.localmemory.org/

What does Local Memory do?

Participatory archiving by local communities and libraries

Why does Local Memory exist?

We see instances of participatory archiving around major, often traumatic community events, such as the Boston Marathon and the 2011 Japan earthquate and tsunami. These are amazing resources built by and for those directly affected.

Could we generalize from their approach, or something like it, to create something that would allow small communities to organize their images, words and ther media around local events, with the help of librarians in the community?

How can I learn more?

If you’re interested in using Local Memory or contributing to the project, check out the project’s Github page or contact us at lil@law.harvard.edu.
