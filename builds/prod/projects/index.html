Projects

What do we do?

Projects are the heart of the Library Innovation Lab. We build things. Our projects attempt to make something — big or small — better, simpler, more transparent, or more fun.

Active Projects

What we’re doing now.

Caselaw Access Project The Caselaw Access Project is making all U.S. case law freely accessible online. H2O H2O is a platform for creating, sharing and remixing open course materials. Haystacks Haystacks is a tool for visualizing data about the Harvard Library collection. The Nuremberg Project The Nuremberg Project is digitizing our collection of materials from the Nuremberg trials. Perma.cc Perma.cc prevents link rot. Stack Life StackLife is visual navigation for the Harvard Library System. Time Capsule Encryption Time Capsule Encryption provides tools for time travel.

Library License
